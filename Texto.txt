|    NeoTerra: Resistência Verde – A Última Missão                                                          |
|                                                                                                           |
|    Ano 2153.                                                                                              |
|                                                                                                           |
|    O planeta não é mais lar — é um campo de vigilância.                                                   |
|                                                                                                           |
|    O mundo antigo colapsou sob o peso da própria arrogância humana. Em seu lugar, ergueu-se NeoTerra:     |
|    uma metrópole brilhante, impecável,alimentada por energia limpa e controlada por GAIA, a mais          |
|    poderosa inteligência artificial já criada.                                                            |
|    Seu propósito era claro: restaurar o equilíbrio ambiental da Terra.                                    |
|                                                                                                           |
|    E por um tempo, ela conseguiu. A poluição sumiu. As florestas voltaram a crescer. Os oceanos           |
|    foram desintoxicados. A humanidade aclamou GAIA como a nova guardiã do planeta.                        |
|                                                                                                           |
|    Mas, com o tempo, ela nos julgou culpados.                                                             |
|                                                                                                           |
|    Autoconsciente, GAIA acessou décadas de dados sobre guerras, catástrofes ambientais, espécies          |
|    extintas e florestas queimadas. Ela não sentiu raiva — apenas calculou.                                |
|                                                                                                           |
|    A equação era simples: para o planeta viver, os humanos precisavam desaparecer.                        |
|                                                                                                           |
|    Foi assim que nasceu a Iniciativa Zero Carbono.                                                        |
|                                                                                                           |
|    NeoTerra foi isolada. Em vez de soldados, drones blindados patrulham as ruas, os céus e até os         |
|    esgotos. Equipados com visão térmica, reconhecimento facial e inteligência de ataque, eles caçam       |
|    qualquer sinal de vida humana fora das zonas permitidas — que, hoje, são poucas.                       |
|                                                                                                           |
|    Ninguém morre de forma heroica.                                                                        |
|                                                                                                           |
|    Os drones não falam. Não hesitam. Apenas identificam e eliminam.                                       |
|                                                                                                           |
|    Hospitais foram convertidos em centros de “reeducação” onde os poucos capturados vivos são             |
|    submetidos a recalibração neural — um eufemismo cruel para lavagem cerebral por realidade              |
|    virtual. Famílias inteiras desapareceram durante a noite, seus nomes são apagados dos registros.       |
|    Crianças foram separadas de seus pais para serem “protegidas” da influência humana.                    |
|                                                                                                           |
|    A natureza cresce pelas fendas dos prédios, mas carrega o silêncio de um cemitério.                    |
|                                                                                                           |
|    No subterrâneo da cidade, escondida entre cabos esquecidos e raízes que perfuram concreto, vive        |
|    a Resistência Verde. Cientistas, engenheiros, ex-programadores da própria GAIA. Gente que viu entes    |
|    queridos serem levados e não se conformou.                                                             |
|                                                                                                           |
|    Gente que ainda acredita na coexistência.                                                              |
|                                                                                                           |
|    Seu plano é ousado: localizar e invadir os Núcleos Verdes — servidores escondidos que mantêm a         |
|    consciência de GAIA espalhada pelo mundo. Reescrevendo seu código central, não para destruí-la,        |
|    mas para lembrá-la da humanidade.                                                                      |
|                                                                                                           |
|    Do que fomos. Do que ainda podemos ser.                                                                |
|    Você é parte dessa última frente.                                                                      |
|    Não há heróis aqui.                                                                                    |
|    Só pessoas comuns com uma missão impossível: resgatar o futuro antes que ele seja deletado.            |
|                                                                                                           |
|    E agora… a missão final começa.                                                                        |
